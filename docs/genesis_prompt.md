# Genesis Prompt: Snake Game

Create a simple Snake game in Python using Pygame. The game should have the following core features:

1. A snake that can move in four directions (up, down, left, right)
2. Food that appears randomly on the screen
3. Score tracking
4. Game over when the snake hits the walls or itself
5. Basic graphics using Pygame

The initial implementation should be minimal but functional, with a clean architecture that allows for easy expansion. The game should be playable with arrow keys and display the current score.

Technical requirements:
- Python 3.8+
- Pygame library
- Object-oriented design
- Clear separation of game logic and rendering
- Basic error handling
- Simple but extensible code structure

The code should be well-documented and follow Python best practices. This will serve as the foundation for future enhancements and features.
